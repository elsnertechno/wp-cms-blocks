== Description ==

Simple and Easy CMS Block which generates Shortcode and can be reuse in any place of the website. Sometimes a same HTML block needs to display multiple places so this plugin really helps. Write content as CMS Block and paste shortcode where you need to be display.

== Installation==

1. Upload `cms-block` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Manage from CMS Blocks in left menu.

== Frequently Asked Questions ==

= How can i add Block in post/page =

1. In WordPress admin panel go to "CMS Blocks", add content in editor and save. Copy the shortcode and put in Page/Post editor

= Can i add other shortcodes in CMS Block editor =

1. Yes.
